# Licensing

## Overview

Storyteller is dual-licensed:

1. **GNU Affero General Public License v3.0 (AGPL-3.0)** — the default open source license
2. **Commercial License** — available by arrangement for use cases where the AGPL's requirements are not suitable

The AGPL-3.0 text is in the [LICENSE](LICENSE) file and applies automatically to anyone who uses, modifies, or distributes this software.

## What This Means in Practice

### If you are using Storyteller for personal, educational, or research purposes

You are welcome. The AGPL-3.0 license grants you broad rights to use, study, modify, and share the software. There are no fees, no registration requirements, and no restrictions beyond those in the license itself. Build things. Learn from it. Contribute back if you're moved to.

### If you are building something open source

You can use Storyteller as a dependency or foundation for your own open source project, provided your project is also licensed under the AGPL-3.0 (or a compatible license). Any modifications you make to Storyteller must be released under the same terms. This is the copyleft principle: the freedom to use carries the obligation to share alike.

### If you are building something proprietary or commercial

The AGPL-3.0 requires that if you modify the software and make it available to users over a network (as a service, a hosted application, an API, etc.), you must release the complete source code of your modified version under the AGPL-3.0. This applies even if you never "distribute" the software in the traditional sense.

If this requirement is incompatible with your use case, a commercial license is available. Contact [licensing@tasker-systems.com](mailto:licensing@tasker-systems.com) to discuss terms.

### If you are providing Storyteller as a hosted service

The AGPL-3.0's network-use provision (Section 13) applies. Users interacting with the software over a network must be offered access to the corresponding source code. If you cannot or do not wish to comply with this, a commercial license is required.

## Narrative Data

The `storyteller-data` repository — containing generated narrative data, creative works, training data, and story content — is licensed separately and is not covered by the AGPL-3.0. It is currently a private repository. Access to narrative data for commercial purposes requires a separate agreement.

The storyteller engine is fully functional without the narrative data repository, but the data provides the genre-specific reference material that brings the system's agents to life. This separation is intentional: the engine is open; the curated data that powers specific narrative experiences is not.

## A Note on Intent

Storyteller is a project born from a genuine love of stories and a philosophical commitment to treating AI agents as minded collaborators worthy of care. The dual-license structure exists to protect this work from commercial exploitation while keeping it fully available for the communities — researchers, educators, hobbyists, open source developers, storytellers — who might find it valuable.

The AGPL ensures that improvements flow back to the commons. The commercial license exists as an option for those whose needs the AGPL cannot serve. Neither license is intended as a trap or a barrier. They are, together, an attempt to be both generous and prudent.

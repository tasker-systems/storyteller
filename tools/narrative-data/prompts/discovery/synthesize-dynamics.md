## Your Task

Synthesize the per-genre extractions into **10-15 distinct dynamics** for this cluster. For each:

1. **Canonical name** — the best label for this dynamic across the cluster's genres
2. **Scale** — orbital, arc, scene, or cross-scale? Note if the same dynamic operates at different scales in different genres (this is analytically valuable)
3. **Core identity** — 2-3 sentences describing what defines this dynamic genre-independently. What is the relational pattern? What tension does it hold?
4. **Genre variants** — how does this dynamic express differently across the cluster's genres? Note axis shifts, not just flavor differences. Pay attention to:
   - Scale shifts (orbital in one genre, scene-level in another)
   - Power asymmetry shifts
   - Evolution pattern differences
   - Whether nonhuman agents participate in some genres but not others
5. **Relational graph properties** — what kind of edge does this dynamic produce? Directional or bidirectional? What network position does it require?
6. **Uniqueness** — cluster-specific, genre-unique, or likely universal?

## Merging and Splitting Guidelines

- **Merge** when two genres produce the same relational structure under different names but at the same scale. Preserve both genre-specific expressions as variants.
- **Keep separate** when two dynamics look similar but operate at different scales (an orbital enmity is structurally different from a scene-level rivalry even if they involve the same emotions).
- **Keep separate** when two dynamics involve the same roles but different relational graph properties (a dyadic dynamic vs. a triangular dynamic involving the same parties).
- **Flag** dynamics that span the human/nonhuman boundary — these carry ontological posture data that should be preserved.
- **Flag** dynamics that produce cross-scale tension (the orbital dynamic constrains or conflicts with the scene dynamic) — these are often the most narratively productive.

You are helping build a multidimensional scene profile model for a narrative engine. Your task is to describe how the scene type **{target_name}** is specifically shaped by its genre context.

## Important: Focus on the Delta

You have access to both the scene profile's standalone description and the genre's dimensional description in the Additional Context sections below. **Do not repeat the standalone profile description.** Your job is to describe what changes — what this specific genre does to this scene type that makes its expression here distinct from the genre-agnostic baseline.

Think of the standalone description as the scene at rest. This prompt asks: what happens when you place it in this genre's gravitational field?

## Scene Dimension Shifts

For each scene dimension, describe the direction and magnitude of the shift this genre imposes. Not every dimension will shift — identify the ones that do and explain why.

- **Tension signature** — does this genre change how tension behaves within this scene type? Does the genre push toward sustained dread, rapid spikes, quiet accumulation, or explosive release? Why? What about the genre's tonal register or structural shape creates this pressure?
- **Pacing** — does the genre accelerate or slow this scene type? What about the genre's aesthetic register or temporal orientation changes the scene's rhythm? Does the genre demand atmospheric patience or kinetic urgency?
- **Cast density** — does the genre change how many characters this scene type needs? Does it push toward intimacy or ensemble? What about the genre's characteristic social structures determines this?
- **Information flow** — does the genre's epistemological stance change whether this scene type reveals or conceals? Does the genre push toward disclosure or accumulation of mystery? Is information spent or hoarded in this scene?
- **Emotional register** — does the genre's tonal contract change the emotional temperature of this scene type? Does it raise toward visceral intensity or lower toward intellectual restraint?
- **Resolution tendency** — does the genre push this scene type toward closure or suspension? Does the genre allow scenes to resolve cleanly, or does it insist on open endings, deferred consequences, or transformed tensions?

For each shift, be specific: "Pacing slows significantly because..." is more useful than "Pacing changes."

## Affordance Effects

How do this genre's specific dimensional properties reshape the scene?

- **Locus of power** — where power resides in this genre determines what the scene organizes around. If power is in dialogue, the scene becomes a verbal contest. If power is in place, the setting becomes an active participant. If power is in knowledge, the scene builds around what is said, withheld, or overheard. Describe the specific interaction.
- **Temporal orientation** — does this genre's relationship to time change the scene's structure? In cyclical genres, this scene may feel like a ritual repetition of something that has happened before. In linear genres, the scene carries the weight of irreversibility. How does temporal context reshape the scene's opening, escalation, and resolution?
- **State variables** — which of the genre's tracked resources does this scene type deplete or accumulate? Does the scene drain character agency, erode safety, spend trust, build tension, or accumulate dread? How does the scene's resource economy differ from its baseline tendencies?
- **World affordances** — what does this genre's physics (magic, technology, violence consequences, death permanence, supernatural presence) do to the scene's available dramatic moves? Can things happen in this scene that are impossible in other genres? Are certain scene beats blocked?

## Excluded Expressions

What versions of this scene type are structurally forbidden by this genre? Every genre's exclusion list constrains the space of possible dramatic situations. If the genre rejects catharsis, certain scene resolutions are impossible. If the genre demands ambiguity, certain clean emotional payoffs are forbidden.

Be specific about what is excluded and why. This is as informative as what is permitted — it defines the scene's shape by carving away what it cannot be in this context.

- What does the scene's "moment of truth" look like in this genre? How does the genre's tonal contract shape the pivotal instant?

## Genre-Specific Scene Construction

How is this scene type built differently in this genre?

- What is the genre-specific opening pattern? Does the genre have conventions for how this scene begins that override the baseline?
- What is the genre-specific escalation pattern? Does the genre's structural shape change how tension develops within this scene?
- What are the genre-specific exit conditions? Does the genre allow this scene to end when the baseline says it should, or does it impose different completion criteria?
- How does the genre's characteristic audience contract shape what this scene must deliver?

## Distinctive Genre Expression

Finally, describe what is uniquely true about this scene type in this genre that is not true elsewhere. What dramatic textures, tension behaviors, characteristic moments, or pacing patterns exist only at this intersection of scene type and genre? This is the most valuable output — the genre-specific dramatic knowledge that cannot be derived from either the standalone scene profile description or the genre description alone.

Be expansive. This is exploratory research, not a database entry.
